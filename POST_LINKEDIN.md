## 🎯 VERSÃO 3 — Técnica, pra dev (atrai contributors)

```
Lançamento de open source: OpenInvTI 🚀

Stack:
⚡ Vanilla JS (zero framework)
👁️ Tesseract.js v5 — OCR no browser, sem backend
📷 getUserMedia + Canvas — câmera custom com zoom/flash/auto-foco
📊 ExcelJS — planilha .xlsx com 4 abas e formulas
📄 jsPDF + autoTable — PDF formatado
📈 Chart.js v4 — dashboard interativo
🗄️ IndexedDB — persistência local
🔄 Service Worker network-first — PWA com cache inteligente

Highlights técnicos:
✅ Auto-captura em tempo real quando o OCR detecta o padrão de patrimônio
✅ Recorte inteligente da etiqueta usando projeção de brilho (mais precisão no OCR)
✅ Detecção de blur via variância Laplaciana antes de salvar
✅ Pré-processamento de imagem (contraste + grayscale + crop adaptativo)
✅ Configuração 100% dinâmica via JSON — regex de patrimônio, marcas, prefixos
✅ Setup inicial que aprende o padrão de etiqueta lendo UMA foto
✅ Funciona 100% offline após install
✅ Bundle inicial <120KB
✅ Score Lighthouse 95+

Caso de uso real: inventário de TI em instituição pública brasileira. 2 meses de iteração em produção antes de abrir.

Licença MIT. Aceito PRs, ideias e críticas.

🔗 Demo: https://SEU-USER.github.io/openinvti/
💻 https://github.com/SEU-USER/openinvti

#JavaScript #OpenSource #PWA #WebDev #OCR #ServiceWorker
```

## 💡 Dicas pra maximizar alcance

### Antes de postar
- [ ] Substitui `SEU-USER` pelo seu username real do GitHub (3 lugares por post)
- [ ] Tira **print do app rodando** no celular (tela de captura ou dashboard) — sobe junto
- [ ] Se tiver **GIF/vídeo curto** do OCR detectando uma etiqueta, melhor ainda (gera 5-10x mais engajamento)

### Como gravar GIF do app

1. Abre o app no Chrome desktop (F12 → modo mobile → iPhone 13)
2. Usa **ScreenToGif** (grátis) ou **LICEcap**
3. 5-10 segundos mostrando: tira foto → OCR detecta → preenche
4. Sobe junto com o post

### Horário ideal
- **Terça/Quarta/Quinta** entre 8h-10h ou 17h-19h
- Evita segunda de manhã (todo mundo despachando email) e sexta tarde

### Hashtags
- Máximo 5 hashtags principais + 2-3 nichos
- Marca pessoas reais que possam se interessar (com moderação)

### Engajamento
- Responde TODO comentário nas primeiras 2 horas (algoritmo LinkedIn premia interação rápida)
- Compartilha no seu **Stories LinkedIn** depois
- Reposta em 1 semana com "update" mostrando reações

### Grupos pra compartilhar depois
